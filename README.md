# Akadem

Simple 3D vehicle physics simulator.

## Controls

+ `W` / `S` - drive forward / backward
+ `A` / `D` - turn left / right
+ `Space` - brake
+ mouse - look around

## How to build

```sh
make
```

### Required libs

+ `SDL`
+ `SDL_ttf`

## 3D builder

Simple tool used to create 3D modeles.

Located in [3d-builder](3d-builder) directory.
